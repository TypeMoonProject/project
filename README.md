# 卷饼餐厅经营游戏

## 游戏简介

一款模拟餐厅经营的策略游戏，玩家需要制作卷饼、薯条和可乐来满足顾客需求，赚取金币并升级店铺设施。

## 游戏特性

• 完整的经营流程：从食材准备到餐品制作的完整链条

• 多样化顾客需求：每位顾客有独特的餐品组合要求

• 库存管理系统：食材和耗材的库存限制与补货机制

• 店铺升级系统：用赚取的金币提升餐厅效率和收益

• 时间挑战：每局有限时间内争取最高收益

## 基本玩法

### 餐品制作流程

卷饼：
1. 放置面饼 → 2. 添加食材（肉/黄瓜/沙司/薯条/番茄酱）→ 3. 卷起 → 4. 包装 → 5. 交给顾客

薯条：
1. 拿薯条盒 → 2. 添加薯条 → 3. 交给顾客

可乐：
1. 拿可乐杯 → 2. 添加可乐 → 3. 交给顾客

### 库存管理

• 直接补货：面饼、黄瓜、沙司、番茄酱、可乐及各种包装材料

• 需要预处理：

  • 肉：切肉后才能补货

  • 薯条：切土豆→炸薯条后才能补货

### 顾客系统

• 顾客有耐心值，会随时间减少

• 需求包含卷饼、薯条、可乐的不同组合

• 卷饼可定制食材搭配

• 必须在耐心值归零前完成所有需求

## 界面说明

### 入口界面

• 开始新游戏

• 店铺升级选项

• 游戏统计信息（已玩局数、总金币）

### 主游戏界面

• 状态栏：天数、金币、剩余时间、餐品价值

• 顾客区域：显示当前顾客需求及耐心值

• 操作台：食材库存和制作中的餐品状态

• 库存显示：各种食材和耗材的当前库存

## 店铺升级选项

1. 自动切肉机：肉库存自动补满
2. 金盘子：提升卷饼售价
3. 扩充店面：同时接待更多顾客

## 操作说明

• 按照正确顺序进行餐品制作操作

• 及时补货维持库存

• 优先处理耐心值低的顾客

• 合理规划多任务并行处理

• 通过不断升级提升餐厅经营效率
